Class Jadva_Tc_Object

Description
Description | Descendents | Vars (details) | Methods (details) Constants (details)

This class represenst an object

A lot of the TC objects link to each other, leading to circular references. This class tries to help, by creating a sort of 'state' for an object. Also adds a few helper functions, like the automatical __get for getVarname functions.
