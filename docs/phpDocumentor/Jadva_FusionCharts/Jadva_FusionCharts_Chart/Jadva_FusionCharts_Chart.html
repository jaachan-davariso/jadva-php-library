		setYaxisMinMaxFix (line 266)
	
 
	
	

Sets whether to apply our y-axis min and max fix


FusionCharts doesn't take trend lines in account, and doesn't account for graphs with only 0 values. We try  to fix that:  
  • If a value contains only 0 values, we set the max y-axis value to 5, ensuring that you don't get weird
       NaN's
  • We recalculate the max and min, and take into account the trend lines
It's off by default. It doesn't work for stacked charts (yet).

	
    
				
  • return: Provides a fluent interface
    • 
				
  • todo: Fix it so this works for stacked charts as well.
