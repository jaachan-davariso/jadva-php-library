Class Jadva_File_Abstract

Description
Description | Descendents | Vars (details) | Methods (details) Constants (details)

Abstract class to represent a file or directory on the file system

Functionality in this class has only been tested on the file system and on vfsStream. Working with relative path is not supported/tested, safe for the realpath function. Also not tested are SMB server or the sort.
